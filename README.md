# Credit Default Risk Prediction

## Project Overview
This project predicts whether a customer is likely to default on a loan using machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Project Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation

## Results
Developed and evaluated classification models to analyze customer credit default risk and improve prediction accuracy.
